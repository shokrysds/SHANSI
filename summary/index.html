خلاصه فصل‌های ۵ تا ۱۰ روانشناسی (۴ صفحه‌ای)

محمد شکری — دوره ۷۴ کارشناسی پرستاری — دانشگاه علوم پزشکی تهران
رفتن به فصل: ۵ | ۶ | ۷ | ۸ | ۹ | ۱۰

فصل ۵: هوش

۴ صفحه اول

هوش یعنی توانایی یادگیری، حل مسئله و سازگاری. استرنبرگ: ظرفیت رفتار هدفمند و سازگارانه. آزمون بینه معرفی «سن عقلی» و فرمول IQ = (سن عقلی ÷ سن تقویمی) × ۱۰۰. مثال‌ها: IQ بالاتر از ۱۰۰ = باهوش‌تر از میانگین.

۴ صفحه دوم

نظریه اسپیرمن (عامل g)، ثرستون (۷ توانایی اصلی)، گاردنر (هوش‌های چندگانه: زبانی، ریاضی، موسیقی... )، گیلفورد (۱۲۰ توانایی).

۴ صفحه سوم

آزمون بینه–سیمون، استنفورد–بینه، آزمون‌های وکسلر (WAIS، WISC، WPPSI) شامل خرده‌آزمون‌ها. آزمون دنور، آزمون‌های گروهی. نمره پایین لزوماً نشانه کم‌هوشی نیست.

۴ صفحه چهارم

عوامل مؤثر: وراثت و محیط (تغذیه، خانواده، آموزش). آسیب‌ها: نابینایی، ناشنوایی، سوءتغذیه. تفاوت‌ها: جنسیتی (پسرها فضایی، دخترها زبانی)، اجتماعی–فرهنگی.

فصل ۶: شخصیت

۴ صفحه اول

شخصیت = الگوی پایدار رفتار و هیجان. نظریه فروید: نهاد، من، فرامن. نئو فرویدی‌ها: آدلر (احساس حقارت)، یونگ (ناخودآگاه جمعی). نظریه صفات (Big Five). آزمون‌ها: پرسشنامه و فرافکن.

۴ صفحه دوم

نظریه تیپ‌ها (مزاج‌ها)، آلپورت (صفات اصلی، مرکزی، ثانویه)، رفتارگرایی (واتسون، اسکینر)، انسان‌گرایی (راجرز، مزلو). عوامل: وراثت + محیط + تجربه‌های فردی.

۴ صفحه سوم

انواع شخصیت: برون‌گرا، درون‌گرا، تیپ A و B. اختلالات شخصیت: گروه A (پارانوئید، اسکیزوئید)، گروه B (مرزی، ضداجتماعی، نمایشی، خودشیفته)، گروه C (اجتنابی، وابسته، وسواسی).

۴ صفحه چهارم

روش‌های سنجش شخصیت: آزمون‌های فرافکن (رورشاخ، TAT)، آزمون‌های عینی (MMPI، NEO). خلاصه: شخصیت = تعامل وراثت و محیط + نظریه‌های متنوع.

فصل ۷: یادگیری و حافظه

۴ صفحه اول

یادگیری = تغییر پایدار در رفتار. شرطی‌سازی کلاسیک (پاولف: زنگ + غذا → بزاق). شرطی‌سازی کنش‌گر (اسکینر: رفتار + پاداش/تنبیه). حافظه: ثبت، ذخیره، بازیابی.

۴ صفحه دوم

حافظه حسی (چند ثانیه)، کوتاه‌مدت (۷±۲ واحد)، بلندمدت (نامحدود). انتقال به بلندمدت نیازمند تکرار و سازماندهی.

۴ صفحه سوم

فراموشی: محو شدن اثر، تداخل پیش‌گستر و پس‌گستر، مشکل بازیابی. عوامل مؤثر: تکرار، معنا‌دار بودن، سازماندهی، علاقه، شرایط جسمی و روانی.

۴ صفحه چهارم

روش PQRST، سازماندهی، تمرین، تصویرسازی، حافظه سازا (بازسازی ذهنی). خلاصه فصل: یادگیری + حافظه = تعامل تجربه و نگهداری اطلاعات.

فصل ۸: انگیزش و پرخاشگری

۴ صفحه اول

انگیزش = نیروی محرک رفتار. نظریه‌ها: غرایز، کاهش سائق، برانگیختگی، مازلو. پرخاشگری = رفتار آسیب‌رسان (کلامی یا جسمی).

۴ صفحه دوم

انواع پرخاشگری: هیجانی (خشم)، ابزاری (هدفمند). عوامل: ناکامی، تبعیض، رقابت. نظریه ناکامی–پرخاشگری: ناکامی → پرخاشگری. واکنش‌ها: اجتناب، واپس‌روی، جابه‌جایی.

۴ صفحه سوم

فرضیه پالایش: تخلیه خشم (همیشه مفید نیست). کاهش پرخاشگری: مجازات درست، آموزش الگوهای آرام، کنترل موقعیت، تقویت رفتار مثبت.

۴ صفحه چهارم

خلاصه: انگیزش نیروی محرک رفتار؛ نظریه‌ها (مازلو، سائق، غرایز). پرخاشگری = رفتار آسیب‌رسان. کنترل: آموزش، جامعه، عدالت اجتماعی.

فصل ۹: تعارض، ناکامی و سازگاری

۴ صفحه اول

تعارض = برخورد اهداف متضاد. انواع: تقرب–تقرب، اجتناب–اجتناب، تقرب–اجتناب، چندگانه. ناکامی = نرسیدن به هدف. سازگاری = توانایی کنار آمدن با شرایط.

۴ صفحه دوم

واکنش‌ها به ناکامی: پرخاشگری، جابه‌جایی، واپس‌روی، گریز روانی، اجتناب، حل منطقی. آزمایش بارکر و همکاران: ناکامی کودکان → پرخاشگری بیشتر.

۴ صفحه سوم

مکانیسم‌های دفاعی: سرکوب، دلیل‌تراشی، فرافکنی، جابه‌جایی، تصعید، واکنش معکوس. اینها اضطراب را کاهش می‌دهند اما مشکل را واقعاً حل نمی‌کنند.

۴ صفحه چهارم

سازگاری سالم: پذیرش واقعیت، کنترل هیجان، حل منطقی، انعطاف‌پذیری. خلاصه: ناکامی و تعارض اجتناب‌ناپذیرند؛ راه درست، سازگاری منطقی است.

فصل ۱۰: مفهوم خود و خودآگاه

۴ صفحه اول

خود = پاسخ به «من کیستم؟». ابعاد: فیزیکی، روانی، اجتماعی، ایده‌آل. عزت نفس = ارزش فرد برای خودش.

۴ صفحه دوم

عوامل شکل‌گیری: مراحل رشد (اعتماد، خودمختاری، ابتکار، کوشش، انسجام نقش). فرهنگ، خانواده، موفقیت و شکست، عوامل زیستی و روانی.

۴ صفحه سوم

خودآگاهی = شناخت افکار، احساسات و تأثیرشان. انواع: خصوصی (درونی) و عمومی (نگاه دیگران). نقش: کنترل رفتار، تصمیم بهتر، رشد فردی، روابط سالم‌تر.

۴ صفحه چهارم

خلاصه: مفهوم خود = تصویر فرد از خودش. عزت نفس = ارزش فرد. عوامل: رشد + فرهنگ. خودآگاهی = توجه به درون و بیرون. اهمیت: کنترل رفتار و رشد.
